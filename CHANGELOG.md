# Changelog

All notable changes to A Tool To Track My Daily Tasks will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2024-12-19

### Added
- Initial project scaffold generated by Project Generator
- Next.js 15 application structure with App Router
- TypeScript configuration with strict mode enabled
- Tailwind CSS setup for styling
- Supabase integration for database and real-time features
- Complete data model with 8 tables:
  - `users` table for user profiles
  - `clients` table for client information and revenue data
  - `projects` table for client projects and campaigns
  - `task_templates` table for recurring task templates
  - `tasks` table for individual tasks with priorities
  - `api_integrations` table for external integration configs
  - `external_task_mappings` table for Asana/ClickUp sync
  - `task_comments` table for task notes and updates
- Database migrations and RLS policies
- Route stubs for 16 pages from site map:
  - `/` - Public homepage
  - `/tasks` - Task list view
  - `/tasks/new` - Create new task
  - `/tasks/:id/edit` - Edit existing task
  - `/clients` - Client management
  - `/clients/new` - Add new client
  - `/clients/:id` - Client detail view
  - `/projects` - Project overview
  - `/projects/new` - Create new project
  - `/templates` - Task templates
  - `/templates/new` - Create new template
  - `/templates/:id/edit` - Edit template
  - `/gmail-integration` - Gmail setup
  - `/settings` - Application settings
  - `/api/tasks` - Tasks API endpoint
  - `/api/gmail/webhook` - Gmail webhook handler
- Integration stubs for external services:
  - Gmail API integration for email-to-task conversion
  - Asana API integration for bi-directional sync
  - ClickUp API integration for team coordination
  - Slack API integration for notifications
- Complete documentation suite:
  - README.md with setup instructions
  - CLAUDE.md briefing file for AI development
  - TECHNICAL_DEBT.md tracking known shortcuts
  - ROADMAP.md with feature timeline
- Environment configuration with example file
- Package.json with all required dependencies
- Git configuration with .gitignore
- Vercel deployment configuration
- ESLint and Prettier configuration
- TypeScript type definitions structure
- Component library foundation with UI components
- Server actions structure for form handling
- Database query layer organization
- Business logic utilities foundation
